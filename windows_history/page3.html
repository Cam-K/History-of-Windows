History of Windows

Before Windows, MS-DOS was the precursor to Windows. DOS only had a command line interface and no graphical user interface of any kind. In the early days of Windows, Windows simply ran on top of MS-DOS, but that is no longer the case. The first version of Windows first came out in 1985 and is drastically different to what we are familiar in Windows today.

Major Windows Releases

  • Windows 1.0 - 1985
  • Windows 2.0 - 1987
  • Windows 3.0 - 1990
  • Windows 95 - 1995
  • Windows 98 - 1998
  • Windows XP - 2001
  • Windows Vista - 2006
  • Windows 7 - 2009
  • Windows 8 - 2012
  • Windows 10 - 2015
